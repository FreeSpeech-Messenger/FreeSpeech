# Contributing to FreeSpeech

We welcome community contributions.

## License Agreement
By contributing, you agree that your contribution will be licensed under the GNU GPL v3.0 (or later).
All commits must include a Signed-off-by line in accordance with the DCO.

## Contribution Workflow
- Fork the repository
- Create a feature branch
- Write clear, documented code
- Add tests if applicable
- Submit a pull request

## Review Process
- All changes require at least 1 maintainer approval.
- Significant architectural changes require community discussion.
- Security-sensitive changes require consensus from core maintainers.

## Governance Model
This project is community-governed.
- Maintainers are selected based on merit and sustained contribution.
- Major decisions are made through open discussion.
- No single entity may unilaterally relicense or privatize the project.

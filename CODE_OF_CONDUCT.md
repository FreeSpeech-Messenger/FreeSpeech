# Code of Conduct

This project adheres to the Contributor Covenant Code of Conduct 2.1.

We are committed to providing a welcoming, harassment-free environment for everyone.

Unacceptable behavior includes:
- Harassment or discrimination
- Personal attacks
- Trolling or bad-faith participation

Maintainers are responsible for enforcing this Code of Conduct.
